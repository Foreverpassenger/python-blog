## 块元素、内联元素、内联块元素

元素就是标签，布局中常用的有三种标签，块元素、内联元素、内联块元素，了解这三种元素的特性，才能熟练的进行页面布局。

**块元素**  
块元素，也可以称为行元素，布局中常用的标签如：div、p、ul、li、h1~h6、dl、dt、dd等等都是块元素，它在布局中的行为：

* 支持全部的样式
* 如果没有设置宽度，默认的宽度为父级宽度100%
* 盒子占据一行、即使设置了宽度

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>块元素</title>
  <style type="text/css">
    body{
      margin:0px;
    }
    
    div{
      background-color:gold;
      text-align:center;
      margin:10px 0;
    }

    .box2{
      width:200px;
      height:60px;
    }

    .box3{
      width:300px;
    }

  </style>
</head>
<body>
 <!--支持全部的样式-->
  <div class="box2">这是第二个div元素</div>
  <!--盒子占据一行、即使设置了宽度-->
  <div class="box3">这是第三个div元素</div>
  <!--如果没有设置宽度，默认的宽度为父级宽度100%-->
  <div>这是一个div元素</div>
</body>
</html>
```

**内联元素**  
内联元素，也可以称为行内元素，布局中常用的标签如：a、span、em、b、strong、i等等都是内联元素，它们在布局中的行为：

* 支持部分样式（不支持宽、高、margin上下、padding上下）
* 宽高由内容决定
* 盒子并在一行
* 代码换行，盒子之间会产生间距
* 子元素是内联元素，父元素可以用text-align属性设置子元素水平对齐方式，用line-height属性值设置垂直对齐方式

**解决内联元素间隙的方法**  
1、去掉内联元素之间的换行  
2、将内联元素的父级设置font-size为0，内联元素自身再设置font-size

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>内联元素</title>
  <style type="text/css">
   body{
      margin:0px;
    }
    .a1{
      width: 200px;
      height: 200px;
      background-color: gold;
      margin-top: 50px;
      margin-bottom: 50px;
      padding-top: 50px;
    }
    .div1{
      width: 300px;
      height: 200px;
      text-align: center;
      margin-top: 20px;
      line-height: 200px;
      background-color: gold;
    }
    .a2{
      background-color: red;
    }
     .div2{
      width: 300px;
      height: 200px;
      margin-top: 20px;
      background-color: gold;
      font-size: 0px;
    }
    .a3{
      background-color: red;
      font-size: 20px;
    }
  </style>
</head>
<body>
    <!--支持部分样式（不支持宽、高、margin上下、padding上下）-->
    <a class="a1" href="#">这是一个内联元素</a>
    <!--宽高由内容决定-->
    <!--盒子并在一行-->
    <!--代码换行，盒子之间会产生间距-->
    <!--宽高由内容决定-->
    <a class="a1" href="#">这是一个内联元素这是一个内联元素</a>
    <a class="a1" href="#">这是一个内联元素这是一个内联元素这是一个内联元素</a>

    <!--子元素是内联元素，父元素可以用text-align属性设置子元素水平对齐方式，用line-height属性值设置垂直对齐方式-->
    <div class="div1">
      <a class="a2" href="#">这是一个内联元素</a>
    </div>
    
    <!--解决内联元素间隙的方法将内联元素的父级设置font-size为0，内联元素自身再设置font-size -->
    <div class="div2">
      <a class="a3" href="#">这是一个内联元素</a>
      <a class="a3" href="#">这是一个内联元素</a>
      <a class="a3" href="#">这是一个内联元素</a>
      <a class="a3" href="#">这是一个内联元素</a>
      <a class="a3" href="#">这是一个内联元素</a>
    </div>
</body>
</html>
```

**内联元素背景高度**

文字高度和背景高度是2种概念，我们这里通过代码可以很明白地看出

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>背景高度</title>
  <style type="text/css">
   body{
      margin:0px;
    }
    .a1{
      background-color: gold;
      padding-bottom: 30px;
    }
  </style>
</head>
<body>
    <!--padding-top和padding-bottom不会影响它的高度，但是会影响他的背景高度,从图中看出红色的div在文字底部的下面，而不是在黄色的底部，这说明了paddingbottom把内联元素的背景高度改变了，但高度仍然是文字的高度-->
    <a class="a1" href="#">这是一个内联元素</a>
    <div style="background-color:red;color: red " >123</div>
</body>
</html>
```

**内联块元素**  
内联块元素，也叫行内块元素，是新增的元素类型，现有元素没有归于此类别的，img和input元素的行为类似这种元素，但是也归类于内联元素，我们可以用display属性将块元素或者内联元素转化成这种元素。它们在布局中表现的行为：

* 支持全部样式
* 如果没有设置宽高，宽高由内容决定
* 盒子并在一行
* 代码换行，盒子会产生间距
* 子元素是内联块元素，父元素可以用text-align属性设置子元素水平对齐方式，用line-height属性值设置子元素垂直对齐方式

这三种元素，可以通过display：inline-block属性来相互转化，不过实际开发中，块元素用得比较多，所以我们经常把内联元素转化为块元素，少量转化为内联块，而要使用内联元素时，直接使用内联元素，而不用块元素转化了。

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>内联块元素</title>
  <style type="text/css">
    .con{
      width:500px;
      height:300px;
      border:1px solid #333;
      margin:50px auto 0;
      font-size:0;
    }

    .con a{
      background-color: gold;
      width:150px;
      height:50px;
      display:inline-block;
      margin:10px 0;
      padding:5px;
      font-size:20px;
      line-height:50px;
      text-decoration:none;
    }

    .con span{
      display:inline-block;
      background-color:gold;
    }


  </style>
  
</head>
<body>
  <!--
  支持全部样式
  如果没有设置宽高，宽高由内容决定
  盒子并在一行
  代码换行，盒子会产生间距(一样通过fontsize解决)
  子元素是内联块元素，父元素可以用text-align属性设置子元素水平对齐方式，用line-height属性值设置子元素垂直对齐方式-->
  <div class="con">
    <a href="#">这是一个a标签</a>
    <a href="#">这是第二个a标签</a>
    <a href="#">这是第4个a标签</a>
    <a href="#">这是第5个a标签</a>
    <a href="#">这是第6个a标签</a>
    <a href="#">这是第7个a标签</a>
    <span>这是一个span标签</span>
  </div>
</body>
</html>
```

**display属性**  
display属性是用来设置元素的类型及隐藏的，常用的属性有：  
1、none 元素隐藏且不占位置  
2、block 元素以块元素显示  
3、inline 元素以内联元素显示  
4、inline-block 元素以内联块元素显示

**内联块元素margin塌陷和margin合并问题**

将将块元素或内联元素转化为内联块元素后，margin塌陷和margin合并都没有发生之前提到的问题

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>内联块元素</title>
  <style type="text/css">
    *{
    margin: 0px;
    padding: 0px;
    }
    .div1{
      width: 200px;
      height: 100px;
      background-color: gold;
      display: inline-block;
    }
    .div2{
      width: 100px;
      height: 50px;
      background-color: red;
      margin-top: 30px;
      display: inline-block;
    }
    .div3{
      width: 800px;
      height: 100px;
      background-color: gold;
      margin-bottom: 30px;
      margin-top: 30px;
      display: inline-block;
    }
    .div4{
      width: 800px;
      height: 50px;
      background-color: red;
      margin-top: 50px;
      display: inline-block;
    }
  </style>
  
</head>
<body>

  <br/>
  <br/>
  <br/>
 
  <!--没有产生margin塌陷解决了-->
  <div class="div1">
    <div class="div2"></div>
  </div>

  <br/>
  <br/>
  <br/>
  <!--没有产生margin合并-->
  <div class="div3"></div>
  <div class="div4"></div>
</body>
</html>
```
